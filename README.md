# Overview

This git repo contains the code a fun side project for me to explore deep learning. I am trying to write a program that classifies if the image was made using a software or if the image comes from a camera.

# Getting Started

## Assumptions

You have the following softwares installed:

- Python 3.14+
- [UV Package Manager](https://docs.astral.sh/uv/)
- [Docker version 20+](https://docs.docker.com/get-docker/)
- Docker Compose

# Training data

Run ```make train``` to train data located in the ```dataset/train``` directory.
